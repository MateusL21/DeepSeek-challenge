Team Task Management API

Bem-vindo à API de gerenciamento de tarefas em equipe

Endpoints disponíveis:

GET / - Esta página
GET /health - Health check da API
POST /auth/register - Registrar novo usuário
POST /auth/login - Login de usuário
GET /teams - Listar times (autenticado)

Para acessar os endpoints JSON, adicione Accept: application/json no header da requisição.
